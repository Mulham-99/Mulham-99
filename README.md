### Security Assessment of Medical Edge Devices
Master’s thesis focused on threat modeling and vulnerability analysis in safety-critical embedded and edge-adjacent systems.

Key areas:
- STRIDE-based threat modeling
- firmware analysis
- network scanning
- fuzzing

### BNSS – Secure Network Architecture
Designed and implemented a secure enterprise-style network environment with:

- WireGuard VPN
- FreeIPA / LDAP
- FreeRADIUS
- DNSSEC
- nftables firewalling
- Snort IDS
- role-based access control
- two-factor authentication

Repository: [BNSS](https://github.com/Mulham-99/BNSS)

Built and hardened a Linux-based self-hosted environment with layered defensive controls.

Included:

- secure remote access
- WireGuard
- port knocking
- CrowdSec
- Suricata

- AppArmor

- TOTP

### Raspberry Pi and OpenWrt Home Infrastructure
Built and maintained a self-hosted Raspberry Pi and OpenWrt-based environment with secure remote access, private DNS, VPN connectivity, firewalling, and containerized services.

Included:
- Pi-hole
- Unbound
- Nextcloud
- Docker
- OpenWrt
- network-level access and privacy controls
